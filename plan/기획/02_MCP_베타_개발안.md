## 1. 개요

### 1-1. 테스트 현황

> 💡 사용하는 LLM 및 에이전트에 따라 답변 표현이 다를 수 있으나, 데이터 조회 결과 자체는 동일하며 큰 차이는 없을 것으로 예상

| AI 플랫폼 | 연결 결과 | BETA 출시 | 설명 |
|-----------|----------|----------|------|
| Kiro | 🟢 성공 | 대상 | AWS AI IDE, MCP 네이티브 지원 |
| Claude Desktop | 🟢 성공 | 대상 | Anthropic 데스크탑 앱, MCP 최초 개발사 |
| ChatGPT | ⚪ 대기 | 대상 | 원격 서버 배포 후 테스트 예정, Pro 이상 플랜 필요 |
| Cursor | 🟢 성공 | 대상 | AI 코드 에디터, MCP 지원 |
| Windsurf | 🟢 성공 | 대상 | AI 코드 에디터, MCP 지원 |
| Gemini | — | 비대상 | CLI 환경에서만 MCP 지원, 일반 웹/앱 미지원 |

### 1-2. 목표

> 💡 모바일인덱스 MCP 서버를 원격(클라우드)에 배포하여, 사용자가 **API 키 + 서버 URL만으로** AI 플랫폼에 연결할 수 있도록 함
> 💡 사용자에게 **연동 가이드 웹페이지**와 **개인 API 키**를 전달, 가이드에 따라 설정하면 AI에서 모바일인덱스 데이터를 자연어로 조회 가능

- 단기 매출 효과는 미미할 것으로 예상. MAU 쏠림이 심하며, MCP 활용을 위해서는 MAU 외 다양한 데이터 활용이 필요
- 단, 프리미엄 고객 한정 기능으로 제공 시 **에센셜 고객의 버전업** 또는 **기존 고객 락인 효과** 기대
- BETA 고객사 섭외는 내부 보안 이슈가 있는 곳만 아니라면 어렵지 않을 것으로 판단

### 1-3. 운영 계획

| 항목 | 내용 |
|------|------|
| 대상 | 기존 고객사 중 BETA 참여 의향 (AM 팀 섭외), 내부 보안 이슈 없는 곳, AI 사용 중인 담당자 우선 |
| 제공 범위 | MCP 서버 전체 27개 도구 (INSIGHT API 전체). 지원: Kiro · Claude · ChatGPT · Cursor · Windsurf |
| 피드백 수집 | 사용 패턴, 불편 사항/개선 요청, 정식 출시 시 유료 전환 의향 |

### 1-4. 사용자 경험

```
➊ 사용자가 개인 API 키를 이메일로 수령
➋ 연동 가이드 페이지 접속
➌ 사용 중인 AI 플랫폼의 MCP 설정에 서버 URL + API 키 입력
➍ 연결 완료 → AI에게 "카카오톡 MAU 알려줘" 같은 자연어 질문으로 데이터 조회
```

### 1-5. 향후 확장 방향

MCP 상품은 **두 가지 데이터 소스**로 개발해야 합니다. 빠르게 출시할 수 있는 API 기반부터 먼저 진행합니다.

| 먼저 — Insight API 기반 MCP | 이후 — 스몰 DMP 기반 MCP |
|----------------------------|------------------------|
| 기존 API 27개를 MCP 도구로 변환 | 스몰 DMP를 만들어 DB 직접 연동 |
| 모바일인덱스에서 볼 수 있는 데이터와 동일 | 모바일인덱스에서 볼 수 없는 데이터 |
| 기획 측 개발 완료, 서버 배포만 남음 | DMP 구축 일정 협의 필요 |

베타는 **Kiro 위주**로 진행하되, 이후 플랫폼별 최적화도 필요합니다.

| 플랫폼 | 베타 | 이후 확장 |
|--------|------|----------|
| Kiro / Cursor / Windsurf | MCP 서버 연결 (현재 방식) | — |
| ChatGPT | MCP 서버 연결 (테스트 후 판단) | **GPT 앱(Actions)으로 별도 개발** |
| Claude | MCP 서버 연결 (현재 방식) | **Claude 커넥터로 개발** |
